AGENTS Instructions
Escopo
Estas instruções se aplicam a todo o repositório agendamento.
Convenções Gerais
Explique de forma clara, nas mensagens finais, o que foi alterado e quais testes foram executados.
Preserve o estilo existente em cada arquivo modificado; observe espaçamento, idioma e formatação local.
Caso uma alteração impacte documentações, atualize os arquivos relevantes.
Quando adicionar novos scripts ou rotinas, inclua instruções de uso básicas se ainda não houver.
Mensagens de PR
Inclua um resumo sucinto das alterações, listado em tópicos.
Documente quais testes foram executados (ou informe se não foram necessários) e seus resultados.
